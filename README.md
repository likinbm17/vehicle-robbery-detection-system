# 🚗 Vehicle Robbery Detection & Management System

## 📌 Overview
A full-stack web application designed to detect and manage vehicle theft cases efficiently.

## 🚀 Features
- Vehicle registration  
- Theft reporting system  
- Real-time tracking  
- Admin dashboard for monitoring  
- DL/LLR approval system  

## 🛠️ Technologies Used
- HTML, CSS, JavaScript  
- PHP, Python  
- MySQL  

## 🎯 Objective
To provide a digital platform for reporting and managing vehicle theft cases in an organized and efficient way.
